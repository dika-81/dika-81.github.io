# Website Pribadi Dr. Andika Arisetyawan, M.Si.

Paket ini dibuat untuk repository GitHub Pages:

`dika-81/dika-81.github.io`

## Isi paket

- `index.html` — halaman utama
- `404.html` — halaman ketika tautan tidak ditemukan
- `favicon.svg` — ikon tab browser
- `CNAME` — mengarahkan GitHub Pages ke `andikaarisetyawan.com`
- `.nojekyll` — memastikan file ditampilkan langsung oleh GitHub Pages

## Cara mengunggah

1. Ekstrak ZIP.
2. Buka repository `dika-81.github.io`.
3. Klik **Add file → Upload files**.
4. Unggah seluruh file yang berada di dalam folder ini.
5. Klik **Commit changes**.
6. Buka **Settings → Pages**.
7. Pada **Build and deployment**, pilih:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
8. Simpan.
9. Pada **Custom domain**, masukkan `andikaarisetyawan.com`.

Catatan: unggah isi foldernya, bukan folder pembungkusnya.
